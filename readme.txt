== Description ==

MusicPress is the plugin for musicians and performers. With MusicPress installed you can easily showcase your tracks and gigs, manage styling, create a fan base, collate bookings and send emails directly from within the WordPress admin. 

Prebuilt with widgets, custom functions and simple [shortcodes] allowing you to enhance your WordPress website. 

MusicPress currently translates in to: 

1. English UK
2. Chinese 
3. Russian
4. French
5. Spanish 

More languages coming with each update.

== Installation ==

1. Upload `musicpress-free` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. To create your tracks and gigs, simply add a new from the MusicPress admin panel.
4. Easily show your tracks or upcoming gigs on a page using the `[mup-tracks]`,`[mup-gigs]` shortcodes. View more functionality on the settings panel.
5. Manage your custom styling through your own stylesheet or through the custom styling tab in the MusicPress main menu.
6. To use the MusicPress widgets, simply drag onto your registered sidebar.


== Frequently Asked Questions ==

= Are there any email restrictions in your plugin? =

Please check with your web host that you can send bulk emails. MusicPress uses `wp_mail()` functionality and maybe restricted by your web host. Pro version (coming very soon) will feature an easier mass email facility.
